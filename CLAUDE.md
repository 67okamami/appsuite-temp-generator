# CLAUDE.md

## プロジェクト概要

AppSuite テンプレートジェネレーター — デスクネッツ ネオの業務アプリ作成ツール「AppSuite」向けテンプレートファイル自動生成ツール。

## ブランチ戦略

- メインブランチ: `master`
- 開発ブランチ: `develop`
- タスクごとにブランチを切り、`develop` に対してPRを作成すること
- ブランチ名は `feature/<タスク概要>` または `fix/<修正概要>` の形式とする

## 作業ルール

### 承認が必要な操作

- **破壊的なコマンド**（`git reset --hard`, `git push --force`, `rm -rf`, ブランチ削除など）を実行する前は、必ずユーザーの承認を得ること
- 設計書の方針に従わない判断をする場合は、理由を述べてユーザーの承認を得ること

### 設計書への準拠

- `.kiro/specs/` 配下の設計書（requirements.md, design.md, tasks.md）に忠実に実装すること
- データモデル、インターフェース、コンポーネント構成、エラーハンドリングは設計書の定義に従うこと
- 設計書に従いたくない場合は、その理由を明確に述べ、ユーザーの承認を得てから代替案で進めること

## 技術スタック

- TypeScript
- Claude API (`@anthropic-ai/sdk`)
- Vitest + fast-check（プロパティベーステスト）
- JSZip（ZIPアーカイブ生成）
- Web Speech API（音声入力）
